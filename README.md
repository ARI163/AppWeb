# 📝 AppWeb - CRUD de Tareas

Aplicación web desarrollada con **Spring Boot** que permite gestionar tareas (crear, listar, editar y eliminar).

---

## 🚀 Tecnologías utilizadas

- ☕ Java 21  
- 🌱 Spring Boot  
- 🗄️ MySQL  
- 🔗 Spring Data JPA  
- 🌐 REST API  

---

## ⚙️ Funcionalidades

✔️ Crear tareas  
✔️ Listar tareas  
✔️ Editar tareas  
✔️ Eliminar tareas  

---

## 📡 Endpoints

| Método | Endpoint        | Descripción        |
|--------|----------------|--------------------|
| GET    | /tareas        | Listar tareas      |
| POST   | /tareas        | Crear tarea        |
| PUT    | /tareas/{id}   | Actualizar tarea   |
| DELETE | /tareas/{id}   | Eliminar tarea     |
